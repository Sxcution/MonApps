# Project Structure - Notes

Rich text notes application with markdown export.

## Files
- `notes_module.py`: Main NotesWidget class with rich text editor
- `debug_ime.py`: Debug tool for IME/Vietnamese typing issues

## Key Features
- Rich text editing (bold, italic, underline, lists)
- Markdown copy option in context menu
- Auto-save notes
- Vietnamese/IME support

## Dependencies
- PySide6
